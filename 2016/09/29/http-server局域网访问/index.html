http-server局域网访问

安装

安装
http-server 是一个简单的零配置命令行HTTP服务器, 基于 nodeJs.
安装 (全局安装加 -g) :

使用
在你所想要的目录下执行命令http-server就可以了。手机和电脑连上同一网络就能实现移动端访问PC端的页面，比较方便，Ctrl+C停止http-server的运行。

关于http-server的用法
官网和网上的教程很多，这里不在细说。

问题

在手机上访问的时候，手机浏览器能够正常访问，当然你关闭了http-server的服务就访问不到了。但是，在微信上访问的时候，却发现了一个问题，那就是微信浏览器的缓存机制，具体是什么我也不太懂。当关闭服务之后，依旧可以访问得到页面，但是这个页面是上一次访问时缓存下来的。现在的问题就是，假如我修改了代码，想要在微信上看下效果，这时候发现却麻烦了，每次访问的都是第一次打开的页面，因为那次被缓存下来了，这就很头疼了….Orz

解决

找到小伙伴讨论此问题之后，找到了解决方案。可以在链接后面加上参数，这就使每一次的访问，都要向服务器重新请求数据和资源，也就更新了页面的内容。这里的请求主要是GET方式的请求，Get是向服务器发索取数据的一种请求，Get请求的参数会跟在url后进行传递，请求的数据会附在URL之后，以?分割URL和传输数据，参数之间以&相连,％XX中的XX为该符号以16进制表示的ASCII，如果数据是英文字母/数字，原样发送，如果是空格，转换为+，如果是中文/其他字符，则直接把字符串用BASE64加密。GET请求的数据会被浏览器缓存起来，所以每一次在后面加上参数可以向服务起发送请求。
例如：http://www.baidu.com?v=456
参数的格式为：链接?参数名称=参数值
此方法针对于微信端，当然普通浏览器也可以，但是普通浏览器没有发现这问题，所以也没必要这样做，完。
