jQuery选择器

May 5, 2016 | FontEnd

简单选择器

在使用jQuery 选择器时，我们首先必须使用“$()”函数来包装我们的CSS 规则。而
CSS 规则作为参数传递到jQuery 对象内部后，再返回包含页面中对应元素的jQuery 对象。
随后，我们就可以对这个获取到的DOM 节点进行行为操作了。

ID选择器

选择器 CSS模式 Jquery模式 描述
元素名 div{} $(‘div’) 获取所有div 元素的DOM 对象
ID #box{} $(‘#box’) 获取一个ID 为box 元素的DOM 对象
类(class) .box{} $(‘.box’) 获取所有class 为box 的所有DOM 对象

子选择器

进阶选择器

群组选择器

CSS模式：span,em,.box {}
JQuery模式: $(‘span,em,.box’)
描述：获取多个选择器的DOM 对象

后代选择器

CSS模式：ul li a{}
JQuery模式:$(‘ul li a’)
描述：获取追溯到的多个DOM 对象

通配选择器

CSS模式：{}
JQuery模式:$(‘‘)
描述：获取所有元素标签的DOM 对象

目前介绍的六种选择器，在实际应用中，我们可以灵活的搭配，使得选择器更加的精准
和快速：

高级选择器

next选择器

CSS模式：div+p{}
JQuery模式:$(‘div+p’)
描述：只获取某节点后一个同级DOM 对象

nextAll选择器

CSS模式：div~p{}
JQuery模式:$(‘div~p’)
描述：获取某节点后面所有同级DOM 对象

补充

在find()、next()、nextAll()和children()这四个方法中，如果不传递参数，就相当于传递
了“*”，即任何节点，我们不建议这么做，不但影响性能，而且由于精准度不佳可能在复杂
的HTML 结构时产生怪异的结果。

为了补充高级选择器的这三种模式，jQuery 还提供了更加丰富的方法来选择元素：

nextUntil()和prevUnitl()方法是选定同级的下面或上面的所有节点，选定非指定的所有
元素，一旦遇到指定的元素就停止选定。
