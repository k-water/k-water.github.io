缘由

为了做一个Java的课程设计，开始接触后端的东西，但是对于后端可谓是一概不知，作为前端er的我还是不成熟。
由于课设要用到后台，又是Java课设，所以就采用了Java。

创建工程

创建JavaWeb Project

File -> New -> Project…

目录结构

Web工程设置

在WEB-INF目录下新建两个文件夹classes和lib，classes主要用来存储编译后的Java文件，lib存放一些jar文件等。

File --> Project Structure...，进入Project Structure窗口，点击 Modules –> 选中项目“JavaWeb” –> 切换到 Paths 选项卡 –> 勾选 “Use module compile output path”，将 “Output path” 和 “Test output path” 都改为之前创建的classes目录

点击 Modules –> 选中项目“JavaWeb” –> 切换到 Dependencies 选项卡 –> 点击右边的“+”，选择 “JARs or directories…”，选择创建的lib目录

效果如图

Tomcat配置

Run -> Edit Configurations
进入“Run Configurations”窗口
点击”+”-> Tomcat Server -> Local
创建一个新的Tomcat容器

在”Name”处输入新的服务名
点击“Application server”后面的“Configure…”
弹出Tomcat Server窗口
选择本地安装的Tomcat目录 -> OK

至此，一个JavaWeb的项目已经大致配置完成
