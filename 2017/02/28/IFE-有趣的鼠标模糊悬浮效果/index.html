边框从两边散开

这个主要是使用了伪元素::before，::after来实现，在父容器上加上伪元素，里面填写的CSS分别的上下，左右的边框，并定位到相应的位置。
在这里父容器使用相对定位，伪元素使用绝对定位，先做好位置。

那么怎么实现边框从两边散开？
在hover的时候，分别吧对应的left 和top值变为0，之前设置了相对父容器的50%。
