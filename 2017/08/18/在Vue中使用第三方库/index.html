在Vue中使用第三方库

Aug 18, 2017 | FontEnd

概述

Vue.js是个很好用很强大的JavaScriptMVVM框架，在实际开发过程中，我们可能还需要引入一些其他的第三方库结合Vue来辅助我们更好的编码，实现需求。

Lodash，Moment，Axios...，这些都是强大有用的JavaScript库，我们可以在Vue很好的使用它们。

但随着我们的项目不断增大，我们需要将代码分离成单个组件和单个模块。我们也希望我们的App能够运行在不同的环境中。

Import方式

另一种方法是在每个文件或组件中import这个库

MyComponent.vue

这种方式不好扩展并且不易维护，甚至可能是痛苦的体验，因为你每使用一次都需要在这个文件import一次，如果你不用了，还要一个个删除它。并且如果你没有设置好你的构建工具（例如webpack），你在最后打包的时候可能会打包多个副本，这无疑大大增加了你打包后代码的体积。
