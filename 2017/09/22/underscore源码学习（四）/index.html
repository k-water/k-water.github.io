排序算法

在JavaScript中，原生ES5为我们提供一个关于数组的排序算法Array.prototype.sort，对于其使用，我们应该相当熟悉。

underscore提供了一个新的排序算法_.sortBy，基于Array.prototype.sort的实现，不过，underscore中的方法功能更加强大，也更加方便。

_.sortBy

_.sortBy提供一个排序依据参数iteratee，该参数经过cb函数进行优化，即根据该比较条件iteratee进行排序。
简单梳理一下_.sortBy的执行过程：

  • 首先通过_.map函数先生成一个新的集合，该集合每个元素都是一个对象，它有三个属性：
    • value: 元素值
    • index: 索引位置
    • criteria: 排序依据，该项通过cb函数优化后计算得到
  • 接着调用原生ES5方法进行排序
  • 最后通过_.pluck函数筛选出value值，过滤掉多余的属性，得到排序的结果

分组

underscore中提供了 一下几个关于分组的API

  • _.groupBy
  • _.indexBy
  • _.countBy
  • _.partition
    前面三个都是基于工厂函数group实现的

group返回一个分组函数，其接受三个参数：

  • obj 待分组对象
  • iteratee 迭代器，通过cb函数优化
  • context 执行上下文

    _.groupBy

    _.groupBy在underscore中的实现

传入group的函数将作为_.groupBy的分组行为函数，当iteratee确定了一个分组后，_.groupBy的行为是

  • 如果分组中存在该分组，将元素push到该分组
  • 如果分组不存在该分组，创建一个新的分组，并将value值放入

注意，如果传入_.groupBy的iteratee不是一个函数而是一个字符串时，那么将使用 iteratee 作为各元素的属性名来对比进行分组。
